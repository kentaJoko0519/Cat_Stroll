# 猫の散歩(Cat-Walk)

### サイトテーマ
ユーザーによって投稿された様々な場所を探すことができる、アウトドア用SNSサイト。

### テーマを選んだ理由
- 旅行やデートなどでどこに行くか悩んだ際に、人気のカフェ、レストラン、観光スポットなどをすぐに知ることができるサイトがあるといいなと考えたから。
- 現在の位置から検索し近くの人気スポットを表示し、投稿したユーザーの評価や、体験したことを知ることができる。

### ターゲットユーザ
- 観光客
- 海外からの観光客
- カップル
- 夫婦
- 家族
- 友人

### 主な利用シーン
- 観光客が観光スポットを探す時
- カップルでデート場所に迷った時
- 友達と遊びに行く時
- 家族で外出、旅行する時
